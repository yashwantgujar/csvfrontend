# 🎯 CSV Upload Dashboard (Frontend)

A modern React dashboard to upload CSV files and track background processing jobs in real-time.

---

## 🖼️ Architecture Overview

![Architecture Diagram](./assets/architecture.png)

> This diagram shows how CSV files are uploaded, processed using Bull Queue, and stored in the database asynchronously.

---

## 🚀 Features

- 📤 Upload CSV file
- ⏳ Real-time job tracking
- 📊 Success & failure stats
- ❌ Row-level error handling
- 🛑 Cancel job
- 📥 Download CSV template

---

## 🛠️ Tech Stack

- React.js
- Axios
- React Toastify
- Bootstrap / MUI
